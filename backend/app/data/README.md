# `app/data/` — bundled data files

## `gaia_bright.parquet`

A bright-star subset of Gaia DR3, used by the SPCC stage to calibrate
colour against real star measurements. Generated by
`backend/scripts/fetch_gaia.py` — see its docstring for details.

Current bundle: **full-sky** G<12 catalogue, 3,083,983 rows, 84 MB
(brotli-compressed Parquet). Fetched by splitting the query into four
RA quadrants to work around ESA's server-side 3M-row cap on async TAP
results.

Schema (Parquet, brotli compression):

| Column              | Type    | Units         | Notes                                 |
|---------------------|---------|---------------|---------------------------------------|
| `ra`                | float32 | degrees ICRS  | ~0.4 arcsec precision at RA=360       |
| `dec`               | float32 | degrees ICRS  | ~0.04 arcsec precision                |
| `phot_g_mean_mag`   | float32 | Gaia G mag    | Main broadband                        |
| `phot_bp_mean_mag`  | float32 | Gaia BP mag   | Blue channel (330-680 nm)             |
| `phot_rp_mean_mag`  | float32 | Gaia RP mag   | Red channel (630-1050 nm)             |

License: Gaia DR3 is released under [CC0 1.0](
https://creativecommons.org/publicdomain/zero/1.0/). Safe to bundle.

### Regenerating

```sh
cd backend
python scripts/fetch_gaia.py          # default: G < 12, auto mirror
GAIA_G_LIMIT=13 python scripts/fetch_gaia.py
GAIA_SOURCE=vizier python scripts/fetch_gaia.py
```

If the ESA Gaia TAP service is down, the script automatically falls
back to the CDS Vizier mirror of the same table.
